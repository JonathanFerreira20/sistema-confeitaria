Projeto de Confeitaria 🍰

Este é um projeto de site de confeitaria desenvolvido com HTML, CSS e JavaScript, criado com o objetivo de apresentar produtos de forma moderna, organizada e interativa. O site possui um design elegante e responsivo, oferecendo uma experiência agradável para os usuários em diferentes dispositivos.

O projeto conta com páginas estilizadas para exibir bolos, doces, sobremesas e produtos personalizados, além de seções como apresentação da confeitaria, catálogo de produtos, avaliações de clientes e contato.

Tecnologias utilizadas
HTML5 → Estrutura do site
CSS3 → Estilização, animações e responsividade
JavaScript → Interatividade e funcionalidades dinâmicas
Funcionalidades
Layout moderno e responsivo
Menu de navegação interativo
Galeria de produtos
Animações suaves ao navegar pelo site
Botões e efeitos visuais dinâmicos
Seção de contato e redes sociais
Objetivo do projeto

O objetivo deste projeto é praticar e aprimorar conhecimentos em desenvolvimento web front-end, trabalhando conceitos de estruturação de páginas, estilização avançada e interatividade com JavaScript, além de criar uma interface visualmente atraente para uma confeitaria.

Resultado

O projeto entrega uma experiência visual agradável, combinando cores suaves, organização e funcionalidades interativas para destacar os produtos e facilitar a navegação dos visitantes.
